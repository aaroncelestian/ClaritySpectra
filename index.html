ClaritySpectra - Raman Spectrum Analysis Tool

A powerful GUI application for importing, analyzing, and identifying Raman spectra.

Installation Guide

Prerequisites

  • Python 3.6 or higher
  • pip (Python package installer)
  • Git (optional, for cloning the repository)

Step-by-Step Installation

1. Set Up Python Environment

First, create and activate a virtual environment if you like, or just use the system python:

python -m venv raman_env
Windows: raman_env\Scripts\activate
macOS: source raman_env/bin/activate
Linux: source raman_env/bin/activate

2. Install Required Packages

Install all required packages using the requirements.txt file:

# Install all required packages pip install -r requirements.txt
Note: For tkinter installation:
  • Windows: Usually included with Python installer
  • macOS: brew install python-tk or reinstall Python with tkinter
  • Debian/Ubuntu: sudo apt-get install python3-tk
  • Fedora: sudo dnf install python3-tkinter

3. Download Required Data Files

Download the following files from Figshare:

  • Raman reference spectra database: https://figshare.com/s/220f7de3c17172dbaae6
  • Raman calculated modes database: https://figshare.com/s/b55e31e89743d246895b

4. Clone the Repository

git clone https://github.com/aaroncelestian/ClaritySpectra.git cd ClaritySpectra

5. Run the Application

python main.py

6. Create a Desktop Shortcut (Optional)

To create a desktop shortcut for easy access:

Windows:
  1. Right-click on your desktop and select "New" → "Shortcut"
  2. In the location field, enter:
    cmd /k "cd /d C:\path\to\ClaritySpectra && python main.py"
    (Replace C:\path\to\ClaritySpectra with your actual installation path)
  3. Click "Next" and name it "ClaritySpectra"
  4. Click "Finish"
macOS:
  1. Open "Script Editor" (in Applications → Utilities)
  2. Enter the following script:
    do shell script "cd /path/to/ClaritySpectra && python3 main.py"
    (Replace /path/to/ClaritySpectra with your actual installation path)
  3. Save as an Application (File → Export → File Format: Application)
  4. Drag the created application to your desktop
Linux:
  1. Create a new file on your desktop named "ClaritySpectra.desktop"
  2. Add the following content:
    [Desktop Entry] Type=Application Name=ClaritySpectra Exec=bash -c "cd /path/to/ClaritySpectra && python3 main.py" Terminal=true Categories=Science;Education;
    (Replace /path/to/ClaritySpectra with your actual installation path)
  3. Save the file and make it executable:
    chmod +x ~/Desktop/ClaritySpectra.desktop
Note: Make sure to replace the example paths with your actual ClaritySpectra installation path. If you're using a virtual environment, you'll need to activate it in the shortcut command.

Key Features

  • Spectrum Import and Processing
    • Import Raman spectra from various file formats (CSV, TXT, DAT)
    • Background subtraction and peak detection
    • Savitzky-Golay smoothing
  • Advanced Analysis Tools
    • Peak fitting with multiple models (Gaussian, Lorentzian, Pseudo-Voigt)
    • 2D Raman map analysis
    • Polarized Raman analysis
  • Experimental Features
    • Crystal orientation analysis and Raman tensor visualization
      Why are these features experimental?

      These advanced features are particularly challenging due to several factors:

      • Complex tensor calculations required for accurate orientation determination
      • Dependency on high-quality polarized Raman data
      • Need for precise crystal structure information (CIF files)
      • Variations in crystal quality and sample preparation
      • Limited database of reference tensor data

      DFT Optimized Raman Spectra:

      • Experimental methods for rapid spectral calculations (< 5 seconds)
      • Uses simplified tensor calculations for quick results
      • Trade-off between speed and accuracy
      • Best suited for preliminary analysis and screening

      Raman Tensor Visualization:

      • 3D visualization of complex tensor properties
      • Integration with crystal symmetry operations
      • Real-time orientation-dependent intensity calculations
      • Limited to supported crystal systems

      These features are under active development and may not provide accurate results for all crystal systems. They are best used for well-characterized, high-quality single crystals.

  • Database Management
    • Mineral database with RRUFF integration
    • Custom database creation and management
    • Metadata management and search
  • Search and Matching
    • Correlation-based search
    • Peak-based matching
    • Machine learning-based search (with TensorFlow)
  • Visualization and Export
    • Interactive plotting with zoom and pan
    • Export results as PDF, CSV, or TXT
    • Generate detailed analysis reports

Getting Started

  1. Launch the application using python main.py
  2. Use the "Import" button to load your Raman spectrum
  3. Process the spectrum using the available tools in the control panel
  4. Search for matches in the database using various search methods
  5. Export your results and analysis reports
Tip: For first-time users, it's recommended to start with the basic spectrum processing features before exploring advanced analysis tools.

Reference Databases

ClaritySpectra integrates with several important Raman spectroscopy databases:

  • RRUFF Project - https://rruff.info/

    Comprehensive database of Raman spectra, X-ray diffraction, and chemistry data for minerals.

  • WURM Database - https://wurm.info/

    Database of calculated Raman modes and crystal structure information.

  • SLOPP/SLOPP-E - https://rochmanlab.wordpress.com/spectral-libraries-for-microplastics-research/

    Spectral Libraries for Organic Pollutants and Plastics Research, including expanded (SLOPP-E) database for microplastics identification.

Important: Make sure to keep your database files up to date and regularly backup your custom databases.
